# cloud-standard-client
졸업프로젝트 cloud-standard의 client 레포지토리입니다.

### cloud-standard는 입문자용 클라우드 서비스입니다.
복잡하고 반복적인 인프라 설정 과정을 추상화하고, 사용 목적에 맞는 표준화된 템플릿을 제공함으로써 사용자가 단 몇 번의 클릭만으로 IaaS(가상머신) 및 CaaS(컨테이너) 환경을 신속하게 구축할 수 있도록 지원합니다.
본 프로젝트의 목표는 기술적 복잡성을 제거하여 사용자가 "이해보다 실행" 중심으로 빠른 성공 경험을 쌓고, 이를 통해 클라우드 기술 활용의 본질인 비즈니스 가치 검증에 집중할 수 있는 환경을 제공하는 것입니다.
